Gunship 2000
------------

Installation:
Um das Spiel zu installieren, erstellen Sie ein Verzeichnis
auf Ihrer Festplatte (z.B. C:\GS2000) und kopieren Sie alle 
Dateien aus dem entsprechenden Verzeichnis auf der CD-ROM
auf Ihre Festplatte.

Um die Einstellungen für das Spiel zu verändern (Joystick,
Soundkarte etc.), wechseln Sie in der MS-DOS-
Eingabeaufforderung in das Spieleverzeichnis auf der
Festplatte und starten Sie das Programm SETUP.

Bitte beachten Sie, daß Sie zum Spielen 580Kbyte 
(594,000 bytes) freien konventionellen Speicher benötigen.


Spielen:
Das Spiel starten Sie, indem Sie in der MS-DOS-
Eingabeaufforderung in das Spieleverzeichnis auf der
Festplatte wechseln und GS2000 aufrufen.
